# grider
> Stylus Grid System using placeholder

### [Visit page for Demo and Instructions](http://diogomoretti.github.io/grider)


## Features
- 100% semantic
- 100% responsive
- Uses placeholders
- Fluid
- Fully-Configurable
- `box-sizing: border-box` included
- Devices included for responsive for all code
